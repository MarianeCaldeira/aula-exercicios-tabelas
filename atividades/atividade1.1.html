Funções das Tags

Table
A etiqueta "table" é usada para definir tabelas.
Th
O elemento HTML "th" define uma célula cabeçalho do grupo de células de sua tabela.
Tr
A etiqueta "tr" é usada para definir uma linha.
Td
A etiqueta "td" é usada para definir uma célula.
Thead
O marcador "Thead", é utilizado para destacar os títulos das colunas de uma tabela.
Tbody
O elemento TBODY estrutura todo o conteúdo da tabela, para que assim o navegador qual exatamente é o conteúdo da tabela.
Tfoot
A tag "tfoot" é utilizada para definir conteúdos de rodapé dentro de uma tabela e juntá-lo ao corpo e ao cabeçalho.
Align
O atributo "align" serve simplesmente para alinhar.
Border
O elemento "Border" é responsável por adicionar bordas em qualquer elemento HTML
Caption
O Elemento HTML "caption" representa o título de uma tabela.
Rowspan
O atributo "rowspan" faz com que a célula ocupe o lugar das próximas células vizinhas abaixo de sua linha.(funde as células na mesma coluna).
Colspan
O atributo "Colspan" faz com que a célula ocupe o lugar das próximas células vizinhas ao lado de sua linha.
Width
A propriedade "width" determina a largura da área de conteúdo de um elemento.
